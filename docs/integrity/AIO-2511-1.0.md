# 🔐 AIO-2511-1.0 — Cryptographic Integrity Proofs

This repository contains cryptographic integrity proofs for audit document **AIO-2511-1.0**. Each chapter is secured using independent Merkle Trees, enabling verifiable authenticity without a central authority.

---

## Purpose

Every sentence is hashed and protected by a Merkle Tree. Any change to the original content can be detected through cryptographic verification.

## Structure

- Every sentence is hashed (SHA-256).
- Hashes are combined into a Merkle Tree.
- The resulting Merkle Root serves as the cryptographic fingerprint of the chapter.
- Complete proof paths are stored for every sentence.

All proofs are stored in `.audit/` as individual JSON files.

## Verification

Run the verification script:

```bash
python3 scripts/verify_audit.py
```
